# mx-icon-money

## Install

```bash
npm install --save mx-icon-money
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'mx-icon-money'
import 'mx-icon-money/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
